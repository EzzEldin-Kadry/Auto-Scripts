For the script to run 
-You need to have all surahs of quraan in the folder_path variable
-You don't have to move any file that is not related to surahs in that folder folder_path unless it's an mp3 file
-All the surah must be in that path, if not, the code will not work properly
-You need to have all surahs names in a txt file and in the txt_path variable
-Every surah name in that txt file must be in a separte line
-This script will change the title in the metadata not the name
